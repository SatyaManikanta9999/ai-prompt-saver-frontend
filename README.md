# AI Prompt Saver

Full-stack AI prompt manager built with:

- React (Hooks)
- Spring Boot
- MySQL
- REST APIs

## Features

- Create, update, delete prompts
- Search functionality
- Dark mode
- Copy to clipboard
- AI platform launcher (ChatGPT, Claude, Gemini, Perplexity)
- Toast notifications
- Validation
- Persistent storage (MySQL)

## Tech Stack

Frontend:
- React
- Fetch API
- CSS

Backend:
- Spring Boot
- Spring Data JPA
- MySQL
- REST API

## How to Run

Backend:
- Run Spring Boot app on port 8080

Frontend:
- npm start (port 3000)
