# AI Prompt Saver

A full-stack AI workflow tool that allows users to:

- Save prompts
- Edit & delete prompts
- Search prompts
- Toggle dark mode
- Copy prompts instantly
- Launch prompts in ChatGPT, Claude, Gemini, Perplexity
- Persistent storage using MySQL

## Tech Stack

Frontend:
- React (Hooks)
- Fetch API
- CSS

Backend:
- Spring Boot 3
- JPA
- MySQL
- REST APIs
- Validation

## Architecture

Controller → Service → Repository → Database
